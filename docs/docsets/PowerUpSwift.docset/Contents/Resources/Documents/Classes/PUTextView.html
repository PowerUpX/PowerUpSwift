Properties

  • newExpandableHeight

    PowerUpSwift: Calculates the new height of the text view based on the size of the content and the user-defined maxExpandableHeight. You should reference an @IBOutlet to the height constraint. Make sure the PUTextView‘s delegate is also set via the Interface Builder or via code.

    Example

    @IBOutlet powerUpTextView: PowerUpTextView!
@IBOutlet powerUpTextViewHeightConstraint: NSLayoutConstraint!

func textViewDidChange(_ textView: UITextView) {
    self.powerUpTextViewHeightConstraint.constant = self.powerUpTextView.newExpandableHeight
}
