PUUnderlineTextField

@IBDesignable
open class PUUnderlineTextField : PUBaseTextField

PowerUpSwift: The subclass of PUBaseTextField that allows the editing of additional UIKit properties via the Interface Builder. This text field displays a bottom bar that appears like an underline.
